Exercícios da quarta aula

  1. Adicione um header nesta página de exercícios, dentro da tag <body/> antes da tag <main/>.
  2. Adicione ao header incluído nesta página:
    • um link que leve para a home do projeto
    • uma imagem dentro do link que leva para a home
    • um menu que tenha os links para cada uma das aulas do projeto
  3. Adicione em cada item de lista do menu, uma nova lista com o link para o exercício das aulas 2, 3 e 4
  4. repita os exercícios 2 e 3 acima para cada uma das páginas criadas no projeto até agora, incluindo a home. Caso a página não tenha a tag <main/>. Adicione-a também à estrtura da página, para manter o padrão das páginas.
