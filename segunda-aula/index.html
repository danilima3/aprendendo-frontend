Relembrando a primeira aula

Para criação do nome de pastas, arquivos e URLs é importante lembrar que:

  • não pode incluir acentuos (`~^) e caracteres especiais, como por exemplo: !@#$%&*
  • sempre substituir ç por c
  • sempre substituir espaço por hífen (-)

Vimos que é possível destacar uma parte do texto colocando em negrito com a tag strong, adicionr um link para outra página com a tag a com o atributo href, titulos com as tags h1, h2, h3, h4, h5 e h6, adicionar um parágrafo com a tag p e adicionar uma lista desordenada com a tag ul com os seus itens definidos pela tag li.

Sempre lembrando de identificar na tag title o título da página em questão para melhor identificação da mesma para os motores de buscas e para os usuários.
